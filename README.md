# Overview
A Python client library for the SkyWise Model API. Currently a WIP.

# Installation

## Prerequisites

- [Python 2.7](https://www.python.org/downloads/)
- [pip](https://pip.pypa.io/en/stable/installing/)

```bash
pip install skywise-model
```
